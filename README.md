share-stuff-community

this project intends to be a plattform where all sorts of goods/stuffs can be shared. Some stuffs like toys for children, not frequently used electronics like boring machines, your nintendo 64, or whatever u want to, can be shared. You can search for stuff near your location und chat with people to make the share happen.
